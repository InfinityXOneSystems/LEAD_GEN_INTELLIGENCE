# XPS Lead Intelligence Dashboard

A modern, responsive dashboard for viewing and managing contractor leads generated by the XPS Lead Intelligence Platform.

## Features

- **Real-time Lead Viewing**: Display contractor leads with scoring and tier classification
- **Advanced Filtering**: Filter by tier (HOT/WARM/COLD), industry, location, and search
- **Dark/Light Mode**: Toggle between themes with electric gold accent colors
- **PWA Support**: Install as a mobile app with offline capabilities
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **GitHub Pages Ready**: Static export for easy deployment

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **UI**: Tailwind CSS 4
- **Language**: TypeScript
- **Deployment**: Static export for GitHub Pages

## Getting Started

### Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build

Build the static export:
```bash
npm run build
```

The output will be in the `out/` directory, ready for deployment to GitHub Pages.

### Data Format

The dashboard expects lead data in the following format at `/data/scored_leads.json`:

```json
[
  {
    "company": "Company Name",
    "phone": "(555) 123-4567",
    "website": "https://example.com",
    "rating": 4.5,
    "reviews": 100,
    "address": "123 Main St",
    "city": "City",
    "state": "ST",
    "email": "contact@example.com",
    "lead_score": 85,
    "tier": "HOT",
    "industry_detected": "Epoxy",
    "rank": 1
  }
]
```

## Deployment

### GitHub Pages

1. Build the static export:
```bash
npm run build
```

2. Deploy the `out/` directory to GitHub Pages

3. Configure your repository settings to serve from the `gh-pages` branch

## Scoring Tiers

- **HOT** (≥75 points): High-quality leads with complete contact info and strong business presence
- **WARM** (50-74 points): Good leads with moderate quality indicators
- **COLD** (<50 points): Basic leads requiring validation

## PWA Features

- Offline support via service worker
- Installable on mobile devices
- Caching for faster load times
- App-like experience on iOS and Android

## Theme Colors

- **Dark Mode**: Pure black background (#000000)
- **Light Mode**: Zinc-50 background (#FAFAFA)
- **Accent**: Electric gold (#EAB308)
